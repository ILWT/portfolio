# Обучение модели классификации комментариев
  
## Описание проекта

Необходимо выделить "токсичные" комментарии к товарам интернет-магазина.

## Навыки и инструменты

- **python**
- **pandas**
- **nltk**
- **scilit-learn**
- tf-idf

## Общий вывод

Для обучения моделей машинного обучения на текстах была произведена их токенизация и лемматизация, тексты очищены от стоп-слов и прочих незначащих символов. Применен TfidfVectorizer для создания мешка слов и расчета TF-IDF (относительной частоты слов в тексте). Из обученных моделей выбрана лучшая (логистическая регрессия) для дальнейшего использования.
